# Pokedex

### Getting started

```sh
npm install
```

##### to run development mode:

```
npm run dev
```
In dev mode at `./dist/main.js` you'll get not minified bundle!
                   
##### to run development mode:

```
npm run build
```
In production mode at `./dist/main.js` you'll get *minified* bundle!

##### to run watch:

```
npm run watch
```
